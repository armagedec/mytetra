Add mytetra to main menu in category 'Education'


vi /usr/share/applications/mytetra.desktop


[Desktop Entry]

Encoding=UTF-8

Version=1.0

Type=Application

Terminal=false

Name=Mytetra

Exec=/home/armagedec/Programs/mytetra/mytetra.run

Icon=/home/armagedec/Programs/mytetra/data/icons/Swift/color_swift_Education.svg

Comment=Memory program

StartupWMClass=Mytetra

Categories=GNOME;GTK;Education;


Add mytetra to startup


vi ~/.config/autostart/mytetra.run.desktop


[Desktop Entry]

Type=Application

Exec=/home/armagedec/Programs/mytetra/mytetra.run

Hidden=false

NoDisplay=false

X-GNOME-Autostart-enabled=true

Name[en_US]=Mytetra

Name=Mytetra

Comment[en_US]=Memory program

Comment=Memory program
