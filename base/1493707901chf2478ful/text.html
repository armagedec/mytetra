Переход в заголовочный файл (#include):

Ctrl + Shift + G


Переключение между открывающими/закрывающими скобочками и т.д.:

Ctrl + ]


Переход к предыдущему/следуюющему активному тексту (который редактировался):

Ctrl + -

Ctrl + Shift + -


Сворачивание блоков (одного и всех):

Ctrl + M, M

Ctrl + M, O


Открытые вкладки:

Ctrl + Tab


Перемещение между вкладками (вперед/назад):

Ctrl + Alt + Pg dn

Ctrl + Alt + Pg up
